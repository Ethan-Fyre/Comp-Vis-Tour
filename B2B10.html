Binary to Base 10 Converter

This page contains a binary to base 10 converter. By inputting a binary value into the form below you can see what the equivalent base 10 value is. In order to calculate this equivalent value, the program will multiply each bit by the corresponding power of 2, and accumulate the results which will equal the base 10 value.

The converter will only properly function if a positive integer is entered. Anything other than an integer will cause the program to refuse to convert the value. It is a picky algorithm, so if you don't treat it right it will not convert.
