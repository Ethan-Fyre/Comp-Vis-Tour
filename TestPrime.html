Testing Primality

This page contains a test for primality. The entered number will be tested agaist all values lower than it to see if it is a composite value, and if it is not, then it is prime.

The primality test will only properly function if a positive integer is entered. Anything other than an integer will cause the program to refuse to test for primality. It is a picky algorithm, so if you don't treat it right it will not function.
